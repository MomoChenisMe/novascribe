## MODIFIED Requirements

### Requirement: Homepage Grid Layout
系統 SHALL 調整首頁網格佈局以適應新的全站 Layout (Sidebar + Content)。

#### Scenario: Responsive grid
- **WHEN** 在桌面版瀏覽首頁
- **THEN** 文章卡片網格應根據剩餘寬度自動調整欄數 (例如 2 欄或 3 欄)
- **THEN** 網格間距應保持一致

### Requirement: Hero Section Adaptation
系統 SHALL 調整 Hero Section 樣式，使其在新的 Layout 結構中視覺平衡。

#### Scenario: Hero display
- **WHEN** 進入首頁
- **THEN** Hero 區塊應填滿內容區域寬度
- **THEN** 圖片與文字比例應適當調整
