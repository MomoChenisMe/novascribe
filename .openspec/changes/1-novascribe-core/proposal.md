# 提案：NovaScribe 核心功能實作 (前台與後台)

## 1. 目標
實作 NovaScribe 部落格系統的核心功能，涵蓋讀者導向的前端介面與創作者導向的後台管理系統，確保在 SEO、效能及 AI 整合方面達到 PRD 3.1、3.2 與 3.3 的要求。

## 2. 範圍

### 3.1 前端 (讀者介面)
- **首頁**：文章網格列表、分頁/無限捲動、分類導覽。
- **文章詳情**：Markdown 高品質渲染、自動目錄 (TOC)、SEO 結構化數據。
- **搜尋**：全域語義搜尋 (AI 驅動)。
- **沉浸式體驗**：沉浸式閱讀模式、PWA 離線支持。

### 3.2 後台 (管理介面)
- **儀表板**：GA 數據可視化、內容健康檢查。
- **內容管理**：Markdown 編輯器、文章版本控制與回溯。
- **媒體管理**：本地/雲端圖片上傳、AI 自動生成 Alt Text。
- **SEO 設定**：全站與單篇文章 SEO 精細化管理。

### 3.3 AI 與性能優化
- 整合向量數據庫 (`pgvector`) 進行相似內容推薦與搜尋。
- 支援 AI 自動摘要與智慧標籤建議。

## 3. 技術棧
- **框架**: Next.js 16.1 (App Router)
- **樣式**: Tailwind CSS
- **資料庫**: PostgreSQL + pgvector
- **認證**: Auth.js (NextAuth)
- **AI**: OpenAI / Gemini / ElevenLabs API
