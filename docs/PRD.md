# NovaScribe (新星筆跡) - 需求規格說明書 (PRD)

## 1. 專案概述
NovaScribe 是一個現代化、高性能且具備極佳 SEO 與 RWD 效能的部落格系統。本系統旨在提供創作者一個直觀、流暢的寫作平台，同時確保讀者在各種裝置上都能獲得最佳的閱讀體驗。

### 1.1 主要目標
- **前後台分離**：提供讀者流暢的閱讀前端與創作者高效的管理後台。
- **SEO 優化**：整合 Google Analytics (GA) 與 Google Search Console (GSC)，並遵循最新 SEO 實踐。
- **響應式設計 (RWD)**：完美適配手機、平板與桌機。
- **高性能**：利用 Next.js 最新技術實現極速載入。

## 2. 技術棧
- **框架**: Next.js 16.1+ (App Router)
- **樣式**: Tailwind CSS
- **資料庫**: PostgreSQL (推薦整合 pgvector 用於 AI 搜尋與推薦)
- **AI 引擎**: 整合 OpenAI / Anthropic / Google Gemini API 或本地 Ollama
- **圖片儲存**: 初始階段存於本地 (Local File System)，預留 Cloudflare R2/Images 擴充介面。
- **部署環境**: Node.js 運行環境 (支援 Vercel / Railway / Self-hosted)

## 3. 功能需求

### 3.1 前端 (讀者介面)
- **首頁**: 文章列表（分頁/無限滾動）、熱門文章推薦、分類導覽。
- **文章詳情頁**: 支援 Markdown 渲染、代碼高亮、目錄導覽、SEO 元數據（JSON-LD）。
- **搜尋功能**: 全文檢索與 **AI 語義搜尋** (基於向量嵌入)。
- **沉浸式閱讀模式**: 類似 Safari 閱讀器，排除干擾。
- **關於我/聯絡頁**: 靜態頁面。
- **[新增] 離線優先 (PWA)**: 支援 Service Workers，讓讀者在斷網環境下仍能閱讀已快取的文章。

### 3.2 後台 (管理介面)
- **儀表板**: 顯示基本的閱讀統計（串接 GA API）與內容健康度總覽。
- **文章管理**: 撰寫（Markdown 編輯器）、編輯、刪除、發布狀態切換。
- **圖片管理**: 上傳圖片至本地資料夾，並產生對應的路徑，支援 AI 自動生成 Alt Text。
- **SEO 設定**: 獨立編輯每篇文章的 Meta Title, Description 與 Open Graph 圖片。
- **網站設定**: GA/GSC 追蹤碼設定。
- **[新增] 內容版本控制與回溯**: 類似 Git 的文章修改歷史，方便追蹤與復原。
- **[新增] 社群互動熱圖 (Heatmap)**: 匿名追蹤讀者閱讀深度與停留點，以熱圖形式呈現給創作者。

### 3.3 創新與 AI 整合功能 (迭代優化)
- **AI 寫作輔助**: 
    - **自動摘要**: 基於文章內容自動生成 Meta Description 與社群分享摘要。
    - **智慧標籤**: AI 自動分析內容並建議適合的分類與標籤。
    - **多國語系翻譯**: 支援一鍵翻譯內容至多種語言，並自動處理各語言版本的 SEO Hreflang 標籤。
    - **AI 寫作風格校閱**: 檢查語法、語氣一致性，並提供精簡或擴充建議。
- **內容增強與轉化**: 
    - **[新增] 語義關聯推薦**: 利用 Vector Embeddings (向量嵌入) 實現精準的文章相關度推薦。
    - **[新增] 內容指紋與版權監測**: 透過 AI 生成內容指紋，監測文章是否被搬運，並強化原創聲明 (JSON-LD)。
    - **[新增] AI 知識庫模式 (Second Brain)**: 讀者可將多篇文章「打包」成 AI 對話上下文進行深度提問。
    - **[新增] 智慧圖片優化**: AI 自動進行 WebP/Avif 轉碼、裁剪，並根據內容自動標註 Tag。
    - **[新增] 互動式數據圖表**: 偵測 Markdown 表格，AI 自動生成可視化圖表 (Recharts)。
    - **社交媒體分發**: 支援將部落格文章自動轉化為社交媒體（如 X/Threads）的短貼文或 Thread。
    - **電子報自動化**: 串接 Resend，根據新文章自動生成週報草稿。
- **SEO 自動化分析**:
    - **內容健康度檢查**: 即時分析文章關鍵字密度、標題結構與連結完整性。
    - **自動優化建議**: 提供改善搜尋排名的建議。
    - **AI 自動化內鏈**: AI 自動識別關鍵字，並建議連結至站內相關文章。
    - **[新增] 關鍵字趨勢預測**: 整合 AI 預測未來熱門關鍵字，建議佈局內容。
- **內容分發與互動**:
    - **自動化 Open Graph 圖片**: 根據文章標題自動生成封面圖片。
    - **語音朗讀 (TTS)**: 整合 AI 語音技術，為每篇文章提供高品質的語音版本。
    - **AI 智能評論助手**: 自動過濾垃圾評論，並提供初步的回覆建議。
    - **互動式 AI 讀書筆記/QA**: 讀者可以針對文章內容直接提問。
    - **[新增] AI 廣告智慧插槽**: 根據語義自動尋找最適合插入贊助連結或 CTA 的位置。
- **[新增] 個人化動態摘要 (Dynamic Snippets)**: 根據讀者的搜尋意圖或過去閱讀偏好，AI 即時調整文章在列表頁的摘要內容，提升點擊率。
- **[新增] 多模態內容轉換**: 支援將 Markdown 內容一鍵轉化為播客腳本 (Podcast Script) 或 短影音腳本 (Reels/TikTok Script)。

## 4. 非功能需求
- **SEO**: 支援動態 Sitemap.xml、Robots.txt、Canonical Tags、Hreflang (多語系)。
- **隱私與合規**: 整合 Cookie 同意管理與 GDPR/CCPA 隱私聲明。
- **性能**: Core Web Vitals 達優良等級，靜態頁面預渲染 (ISR) 優化速度。
- **安全性**: 完善的身份驗證 (Auth.js) 與輸入驗證。
- **[新增] 永續性 (Digital Sustainability)**: 優化渲染效能降低碳足跡，並提供「節能模式」介面。
- **[新增] 去中心化身分與內容證明 (Web3/DID)**: 整合去中心化標識符 (DID) 或雜湊上鏈 (如 Arweave/IPFS)，確保內容在 AI 幻覺氾濫的時代具有不可篡改的實人創作證明。
